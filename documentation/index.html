Overview

Php Form Builder is a simple but powerfull class in witch you can create complex forms easily.

Form is ready to use with Bootstrap, but can be easily adapted for any framework or be used as standalone.
All you need is to match form options with your framework.

Form generates clean HTML5 markup, with two separates functions to generate CSS and JS code, so you can render them separately (generaly in <head> or just before </body>).

Server-side validation can be done on any destination page ; errors are stored in session

Email sending function : send simple email based on default html/css template and filter posted values, or advanced email with custom template, attached files, cc, bcc, etc.

Many plugins included (colorpicker, datepicker, timepicker, captcha, word/character counter, file upload, tinyMce, country select, ...), .
You can easily add any other external plugin.

Add any plugin to your form with just one line of code. Plugins configurations are saved, needed js & css files will be automatically called.

 

How to start ?

  • Go to Quick Start Guide to create your first form
  • Go to PHP Form Builder Documentation to add fields, plugins, customize layout, group inputs, add custom html, ...
  • Go to Examples to see various prebuilt templates : complete forms including Modal forms, Ajax forms, Step forms, or single fields usage examples.

Quick Start Guide

  1. Copy phpformbuilder folder on your server :

    phpformbuilder/

    database/* not required if you don't have to connect any db *
    mailer/* not required if you don't send any email *
    plugins/* not required if you don't use any plugin *
    plugins-config/* not required if you don't use any plugin *
    Validator/* required *
    Form.php* required *
    plugins-path.php* not required if you don't use any plugin *

    plugin/ and plugins-config/ dirs are required only with use of plugins.
    If your form uses plugins, configure plugins path in plugins-path.php.

  2. configure plugins path in plugins-path.php

  3. Start session and include form class :

    <?php
use phpformbuilder\Form;
use phpformbuilder\Validator\Validator;

session_start();
include_once('form/Form.php');

  4. Instantiate your form :

    $form = new Form('my-first-form'); // minimal ; default layout will be 'horizontal'

    or

    $form = new Form('my-first-form', 'vertical'); // layout can be horizontal | vertical | inline

  5. Add content, for example :

    $form->startFieldset('My First Form');
$form->addInput('text', 'my-input-name', '', 'Your name', 'required=required');
$form->addTextarea('my-textarea-name', 'Your message :', 'Your message : ', 'cols=30, rows=4');
$form->addBtn('submit', 'my-btn-name', 1, 'Submit form', 'class=btn btn-primary');
$form->endFieldset();
$form->render();

Features

  • Form generation with options for layout (horizontal, vertical, inline).
  • Can build ANY form in ANY situation (simple form, modal, ajax, step form, ...)
  • Bootstrap layouts friendly (rows, cols, input groups, button groups, ...)
  • Accepts any HTML5 form elements, including fieldsets, multiple selects, optgroups, button groups, ... .
  • Accepts any javascript logic (dependant fields, jQuery plugins, Bootstrap js, ...) with no limit.
  • Allows to customize HTML with wrappers, IDs, classes and attributes, Javascript events, custom code anywhere.
  • Default options ready for Bootstrap.
  • Renders separate HTML code, CSS includes code for plugins, JS includes code for plugins and JS Domready code for plugins.
  • Server-side validation, with multilanguage errors gestion (Third part PHP class).
  • XSS, CSRF protected
  • Send posted values by email based on customisable templates.
  • Record posted values into database.
  • Can display HTML code for debug purpose.
  • Powerful integration of any Javascript plugin (jQuery or any other library) : configure the plugin XML file, PHPForms will generate all code for you (CSS includes, JS includes and JS Domready).
  • Can memorize several custom configurations with each plugin (as many as you want), so you can reuse them easily.
  • Ready-to-use jQuery plugins included :
    • Colorpicker
    • Datepicker
    • Timepicker
    • Captcha
    • word/character counter
    • Fileupload
    • TinyMce
    • Bootstrap Select
    • Country Select (multilanguage with flags)
  • Many Forms Templates.
  • Single Fields Templates.
  • Complete documentation.
  • Support on Codecanyon, by e-mail or by phone.
